Simulation¶

The simulation module initialized the biosim object, and has methods for starting the simulation, setting parameters in cell and fauna, adding population to the cells. It can also get properties about the simobject and save results if needed.

The simulation module¶

class biosim.simulation.BioSim(island_map, ini_pop, seed, ymax_animals=None, cmax_animals=None, img_base=None, img_fmt='png', save_csv=False)[source]¶

The BioSim class handels the creation of a biosim-object and methods for simulation, adding population, setting landscape- and animalparameters. There is also a method for making an MPEG4 movie from saved simulation images.

Add a population to the island based on dictionary containing location(‘loc’) and population(‘pop’). ‘pop’ contains species, age, and weight of the creature. :param population: List of dictionaries specifying population

static check_validity_of_string(island_map)[source]¶

This function checks if the mapstring is valid according to the project specifications, that is Ocean cell at the map’s edges and no other landscape than Ocean, Jungle, Savannah, Desert and Mountain, and returns a boolean. :param map_string: multiline-string :return: boolean
